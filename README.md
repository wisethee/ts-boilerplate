Webpack 5 boilerplate using TypeScript and Sass

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:8080`.

### Production build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```
